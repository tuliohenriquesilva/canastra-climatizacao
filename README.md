# ❄️ Canastra Climatização

> Website institucional moderno e responsivo desenvolvido para a **Canastra Climatização**, especializada em soluções de conforto térmico residencial e comercial. O projeto destaca-se pela experiência visual fluida e interativa, inspirada em animações de grandes marcas de bebidas (como o efeito scroll da Fanta/Coca-Cola).

---

## 💡 A Ideia do Projeto

A intenção principal foi fugir dos sites institucionais estáticos e tradicionais. Para isso, aplicamos conceitos avançados de manipulação de scroll, onde elementos visuais chaves como a **unidade de ar-condicionado** e **pedras de gelo** flutuam e se movimentam de forma sincronizada conforme o usuário navega pelas seções.

### 🚀 Principais Funcionalidades
* **Animação Premium de Rolagem:** Movimentação e rotação suave de elementos usando GSAP.
* **Efeito Cascata (Stagger):** Cards de serviços surgem de forma sequencial e elegante.
* **Interface Limpa e Fria:** Paleta de cores baseada em azul escuro e azul claro, transmitindo tecnologia e frescor.
* **Foco em Conversão (CTA):** Botões estratégicos direcionando o cliente direto para o WhatsApp de atendimento.
* **Totalmente Responsivo:** Layout adaptado para smartphones, tablets e desktops.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais combinadas com bibliotecas de alta performance para animação:

* **HTML5:** Estrutura semântica e organizada.
* **CSS3:** Estilização moderna, Flexbox, Grid Layout, variáveis (`:root`) e efeitos de flutuação nativos (`@keyframes`).
* **JavaScript (ES6):** Lógica de interação do cabeçalho e controle de eventos.
* **GSAP (GreenSock Animation Platform):** Core das animações de alta performance.
* **ScrollTrigger Plugin:** Sincronização milimétrica das animações com a rolagem do navegador.
* **Font Awesome:** Biblioteca de ícones vetoriais.
